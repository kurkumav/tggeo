## 📝 Что нужно сделать:

### Шаг 1: Создайте репозиторий на GitHub

1. Откройте https://github.com/new
2. Название репозитория: **tggeo**
3. Выберите **Public**
4. НЕ добавляйте README, .gitignore (у нас уже есть)
5. Нажмите **Create repository**

### Шаг 2: Загрузите index.html

**Вариант A: Через веб-интерфейс (проще)**

1. На странице нового репозитория нажмите **uploading an existing file**
2. Перетащите файл `index.html` из папки `C:\tggeo\`
3. Нажмите **Commit changes**

**Вариант B: Через Git (если установлен)**

```bash
cd C:\tggeo
git init
git add index.html
git commit -m "Add WebApp"
git branch -M main
git remote add origin https://github.com/kurkumav/tggeo.git
git push -u origin main
```

### Шаг 3: Включите GitHub Pages

1. В репозитории перейдите в **Settings** (вверху справа)
2. Слева найдите **Pages**
3. В разделе **Source** выберите **main** branch
4. Нажмите **Save**
5. Подождите 1-2 минуты

### Шаг 4: Проверьте что WebApp работает

1. Откройте в браузере: https://kurkumav.github.io/tggeo
2. Должна показаться страница с картой
3. Подождите - должно загрузиться фото

### Шаг 5: Запустите бота

```bash
cd C:\tggeo
python bot.py
```

Должно появиться:
```
INFO:__main__:Бот запущен...
```

### Шаг 6: Протестируйте!

1. Откройте Telegram
2. Найдите вашего бота (по username который создали у @BotFather)
3. Отправьте `/start`
4. Нажмите "🎮 Играть"
5. Играйте!

## 🐛 Если что-то не работает:

### WebApp не открывается
- Подождите 2-3 минуты после включения GitHub Pages
- Проверьте что `index.html` загружен в репозиторий

### Бот не отвечает
- Проверьте что `python bot.py` запущен
- Посмотрите логи в консоли

### Фото не загружается
- Откройте консоль браузера (F12)
- Посмотрите на ошибки
