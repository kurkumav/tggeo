# ✅ Чеклист запуска TGGeo

Используйте этот чеклист чтобы убедиться что всё настроено правильно.

## 📝 Подготовка (5-10 минут)

- [ ] **Установлен Python 3.8+**
  - Проверка: `python --version`
  - Если нет: скачать с python.org

- [ ] **Получен Telegram Bot Token**
  - Открыть @BotFather в Telegram
  - Отправить `/newbot`
  - Скопировать токен (формат: `1234567890:ABCdef...`)

- [ ] **Получен Mapillary API Token**
  - Зарегистрироваться на mapillary.com
  - Перейти в Dashboard → Developers
  - Создать приложение
  - Скопировать Client Token (формат: `MLY|1234567890|abc...`)

- [ ] **Создан аккаунт на GitHub**
  - Если нет: зарегистрироваться на github.com

## 🔧 Настройка проекта (5 минут)

- [ ] **Скачаны все файлы проекта**
  - bot.py
  - index.html
  - requirements.txt
  - .env.example
  - .gitignore

- [ ] **Установлены Python зависимости**
  ```bash
  pip install -r requirements.txt
  ```
  - Должно установиться без ошибок

- [ ] **Создан файл .env**
  ```bash
  cp .env.example .env
  ```
  - На Windows: скопировать через проводник

- [ ] **Заполнен файл .env**
  - `TELEGRAM_BOT_TOKEN=` ваш токен от BotFather
  - `MAPILLARY_TOKEN=` ваш токен от Mapillary
  - `WEBAPP_URL=` пока оставить пустым

## 🌐 Размещение WebApp (5-10 минут)

- [ ] **Создан репозиторий на GitHub**
  - Название: `tggeo` (или любое другое)
  - Тип: Public

- [ ] **Загружен index.html**
  - Через веб-интерфейс: "Add file" → "Upload files"
  - Или через Git: `git add index.html && git commit && git push`

- [ ] **Включен GitHub Pages**
  - Settings → Pages
  - Source: main branch
  - Save

- [ ] **Получен URL GitHub Pages**
  - Подождать 1-2 минуты
  - URL вида: `https://username.github.io/tggeo`
  - Открыть в браузере - должна показаться карта

- [ ] **URL добавлен в .env**
  - Открыть `.env`
  - Вставить: `WEBAPP_URL=https://username.github.io/tggeo`
  - Сохранить

## 🚀 Запуск (1 минута)

- [ ] **Запущен бот**
  ```bash
  python bot.py
  ```
  - Должно появиться: "Бот запущен..."
  - Окно не закрывать!

## ✅ Тестирование (2 минуты)

- [ ] **Найден бот в Telegram**
  - Поиск по username (который создали у BotFather)

- [ ] **Отправлена команда /start**
  - Бот должен ответить с кнопкой "🎮 Играть"

- [ ] **Открыт WebApp**
  - Нажать "🎮 Играть"
  - Должна открыться страница с фото и картой

- [ ] **Фото загрузилось**
  - Подождать 2-3 секунды
  - Должно показаться фото улицы

- [ ] **Выбрано место на карте**
  - Кликнуть на карту
  - Должен появиться красный маркер

- [ ] **Отправлен результат**
  - Нажать "🎯 Угадать"
  - Бот должен прислать результат с расстоянием и очками

## 🎉 Готово!

Если все пункты отмечены - ваш бот работает!

## 🔍 Если что-то не работает

### Бот не отвечает на /start
→ Проверьте что `bot.py` запущен и показывает "Бот запущен..."
→ Проверьте `TELEGRAM_BOT_TOKEN` в `.env`

### WebApp не открывается
→ Проверьте `WEBAPP_URL` в `.env`
→ Откройте этот URL в браузере - должна быть карта
→ Подождите 2-3 минуты после включения GitHub Pages

### Фото не загружается
→ Проверьте `MAPILLARY_TOKEN` в `index.html` (строка 18)
→ Откройте консоль браузера (F12) и посмотрите ошибки

### Ошибка при установке зависимостей
→ Обновите pip: `pip install --upgrade pip`
→ Попробуйте снова: `pip install -r requirements.txt`
