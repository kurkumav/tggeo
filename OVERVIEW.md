# 🎮 TGGeo - Краткое описание

## Что это?
Telegram бот-игра в стиле GeoGuessr. Показывает случайное фото из реального мира, игрок угадывает место на карте.

## Файлы проекта

```
tggeo/
├── 📄 README.md          - Полная документация с пошаговыми инструкциями
├── 📄 QUICKSTART.md      - Быстрый старт (5 минут)
├── 📄 FAQ.md             - Часто задаваемые вопросы
├── 📄 CLAUDE.md          - Техническая документация для AI
│
├── 🐍 bot.py             - Telegram бот (Python)

│
├── ⚙️ requirements.txt   - Python зависимости
├── ⚙️ .env.example       - Шаблон конфигурации (скопировать в .env)
└── ⚙️ .gitignore         - Git ignore файл
```

## Быстрый старт (3 шага)

### 1️⃣ Получите токены
- **Telegram Bot**: @BotFather → `/newbot`
- **Mapillary API**: mapillary.com/dashboard/developers

### 2️⃣ Настройте проект
```bash
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env - вставьте токены
```

### 3️⃣ Разместите и запустите
- Загрузите `index.html` на GitHub Pages
- Вставьте URL в `.env` как `WEBAPP_URL`
- Запустите: `python bot.py`

## Архитектура

```
Пользователь → Telegram Bot → index.html (GitHub Pages) → Mapillary API
                    ↓
              bot.py (Python)

            Результат игры
```
